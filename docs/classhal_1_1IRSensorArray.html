Detailed Description

Array de sensores IR conectados aos canais ADC do RP2040.

Converte leituras para faixa [0..1], onde 0 ~ preto/baixa reflexão e 1 ~ alta reflexão.

Constrói o array de sensores IR, inicializando ADC e GPIOs associados.

Parameters
adc_leftíndice do canal ADC do sensor esquerdo (0..3, 26..28 se via GPIO mapping)
adc_frontíndice do canal ADC do sensor frontal
adc_rightíndice do canal ADC do sensor direito

Canais válidos do ADC RP2040 são 0..4; quando 0..3, mapeiam para GPIO26..29. GPIOs correspondentes são configurados para função ADC automaticamente.

Define o fator de suavização exponencial (EMA).

alpha in (0,1]; valores menores suavizam mais. Ex.: para dt=150ms e tau~0.5s: alpha≈dt/(tau+dt)≈0.23.
