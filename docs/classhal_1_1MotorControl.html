Detailed Description

Controla dois motores (esquerdo e direito) de tração diferencial.

Responsabilidades:

  • Inicializar GPIOs/pinos e periféricos (ex.: PWM) conforme a implementação.
  • Aplicar comandos de velocidade normalizada com saturação e direção.
  • Oferecer modo arcade (avançar/rotacionar) para navegação simples.

Thread-safety: não é intrinsecamente thread-safe. Se houver acesso concorrente a partir de ISRs/threads, proteger externamente (mutex/desabilitar IRQs).
